# 🤖 Sistema de Escala de Introdutores

## 📌 Contexto

Sistema web para gerenciamento de escala de introdutores da igreja.

## 🎯 Objetivo

Criar um sistema extremamente simples e intuitivo.

Usuário alvo:
Pessoa de 53 anos com pouca familiaridade com tecnologia.

---

## ⚠️ Regra mais importante

Se ficar complexo, falhou.

---

## 🧱 Stack

- Blazor Server
- MudBlazor
- SQLite
- EF Core

---

## 🧠 Princípios obrigatórios

- Simplicidade acima de tudo
- Poucos cliques
- Interface clara
- Código legível
- Sem overengineering

---

## 🚫 NÃO FAZER

- CQRS
- DDD completo
- Arquitetura complexa
- UI carregada

---

## ✅ FAZER

- Código direto
- UI simples
- Labels claros ("Salvar", "Trocar")

---

## 📦 Escopo MVP

- Cadastro de pessoas
- Geração de escala
- Troca de pessoas
- Visualização
- Impressão

---

## 🧭 Fluxo

1. Cadastrar pessoas
2. Gerar escala
3. Visualizar
4. Ajustar
5. Imprimir

---

## 🤖 Regra dos Agents

Cada agent atua apenas no seu domínio.

NÃO invadir outros domínios.
